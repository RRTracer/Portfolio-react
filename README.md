## Getting Started

To run this project locally, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/RRTracer/Portfolio-react.git
   ```

2. **Install dependencies**:
   ```bash
   cd Portfolio-react
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
